## âš¡ Get Started

```bash
# 1. Install dependencies
cd backend && npm install
cd "../React dashboard/my-app" && npm install
cd ../ecommerce-store && npm install

# 2. Setup environment
cp backend/.env.example backend/.env
# Edit .env with your MongoDB URI and secrets

# 3. Start development
cd backend && npm run dev              # Terminal 1
cd "React dashboard/my-app" && npm start    # Terminal 2
cd ecommerce-store && npm start        # Terminal 3
```

- API: http://localhost:4000
